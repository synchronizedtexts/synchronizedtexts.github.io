German-English Synchronized Texts

To help English readers learn to read German, Synctexts offers synchronized dual-language (German-English) versions of public domain (at least in the United States) German stories and literature. The new English translations are by Synctexts and are intended, by sticking very close to the original German texts, to help those just getting started with the German. For this reason, the translations have deliberately not been rendered into the most natural-sounding English prose; the English translations here are for learning German, not for learning English. I have generally made concessions to English-language sensibilities only when the English would have turned out to be exceptionally awkward.

The full versions of the works use dynamic synchronization of the texts; these versions require a browser with robust JavaScript and strong CSS support. Other, more accessible, versions use static synchronization of the texts and do not make use of JavaScript and do not require CSS; these versions are marked with "alt" in the links below. There are also tooltip versions which use static synchronization via the HTML title attribute (which some browsers display as a tooltip during a mouseover action); these tooltip versions (marked 'tip') use JavaScript and CSS but do not require their use.

The works are roughly arranged, top to bottom, in order of difficulty, those toward the top generally less challenging than those toward the bottom.
